Matrix Factorization Movie Recommender

Trains a Matrix Factorization model (user/item embeddings + dot product) entirely in your browser with TensorFlow.js. Then predicts a rating for a selected User × Movie.

Read the explanation first (why this is correct & how to review)
  1. Model: ŷ = dot(userVec, movieVec) + userBias + movieBias (linear output).
  2. Training: MSE loss, Adam(0.001), shuffled mini-batches, validation split.
  3. ID mapping: raw IDs → contiguous indices required by embeddings.
  4. Limits: cold users/items have no embeddings; ratings are 1–5 (we clamp for display).
